Parsers
*******************

The following parsers are provided to deal with various RPC responses from the radio:

 * An :doc:`events parser <parsers/events>` for parsing the response for the *get-events* RPC.
 * A :doc:`get parser <parsers/get>` for parsing the response for the *get* RPC.
