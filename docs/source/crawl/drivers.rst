Drivers
*******************

The following drivers are provided to actually communicate with a TG radio:

 * An :doc:`SSH NETCONF driver <drivers/netconf>` for all NETCONF RPCs, as well as for tunneling into remote (IP-less) Client Nodes.
 * A :doc:`Tech-Log driver <drivers/techlog>` for fetching the tech log via HTTP.
